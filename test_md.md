## 4. 数据输出

### 4.1 输出文件列表

```
test_output/
├── test_results.json          # 测试结果 (JSON)
├── test_results.csv           # 测试结果 (CSV)
├── simulation_data.json       # 仿真数据 (JSON)
├── simulation_data.csv        # 仿真数据 (CSV)
└── plots/
    ├── tissue_energy.dat      # 组织能量数据
    ├── suture_length.dat      # 缝合线长度数据
    ├── suture_motion.dat      # 缝合线运动数据
    ├── instrument_motion.dat  # 器械运动数据
    ├── performance.dat        # 性能数据
    └── plot_script.gnuplot    # Gnuplot绘图脚本
```

### 4.2 可视化数据

#### 组织能量曲线
- 文件: `plots/tissue_energy.dat`
- 格式: 时间(s) 能量(J)
- 描述: 记录组织总能量随时间变化

#### 缝合线长度曲线
- 文件: `plots/suture_length.dat`
- 格式: 时间(s) 长度(m)
- 描述: 验证约束保持效果

#### 缝合线运动曲线
- 文件: `plots/suture_motion.dat`
- 格式: 时间(s) Y位置(m)
- 描述: 质心在重力作用下的运动

#### 器械运动曲线
- 文件: `plots/instrument_motion.dat`
- 格式: 时间(s) X位置(m)
- 描述: 恒力作用下的运动轨迹

#### 性能曲线
- 文件: `plots/performance.dat`
- 格式: 时间(s) 计算时间(ms)
- 描述: 每帧计算耗时

### 4.3 绘图方法

使用Gnuplot生成图表：

```bash
cd test_output/plots
gnuplot plot_script.gnuplot
```

将生成以下PNG图像：
- `tissue_energy.png` - 组织能量曲线
- `suture_length.png` - 缝合线长度曲线
- `suture_motion.png` - 缝合线运动曲线
- `instrument_motion.png` - 器械运动曲线
- `performance.png` - 性能曲线
