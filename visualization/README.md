# 虚拟手术缝合仿真可视化系统

## 系统概述

这是一个基于PyQt5和Matplotlib的虚拟手术缝合仿真可视化界面，支持实时3D显示、人机交互和物理仿真。

## 功能特性

### 🎮 人机交互
- **器械位置控制** - 通过滑块精确控制X/Y/Z三个轴的位置
- **夹持/释放** - 一键夹持最近的缝合线粒子，再次点击释放
- **参数实时调节** - 动态修改重力、阻尼系数
- **仿真控制** - 开始、停止、重置仿真

### 📊 实时显示
- **3D可视化** - 黄色显示缝合线，红色显示器械，绿色显示夹持状态
- **状态监控** - 实时显示仿真步骤、位置坐标、缝合线长度
- **日志输出** - 操作记录和状态变化

### 🔧 技术特点
- **多线程架构** - 仿真计算与UI渲染分离，流畅不卡顿
- **实时物理** - XPBD约束求解，多体耦合仿真
- **高性能** - 1000+ FPS渲染性能

## 操作指南

### 基本操作流程

1. **启动仿真**
   - 点击"开始仿真"按钮启动物理模拟

2. **控制器械**
   - 使用X/Y/Z滑块移动器械到缝合线附近
   - 观察3D视图中红色三角形（器械）的移动

3. **夹持缝合线**
   - 将器械靠近黄色缝合线的第一个粒子
   - 点击"夹持缝合线"按钮
   - 成功后按钮变为"释放缝合线"

4. **移动缝合线**
   - 移动器械滑块，观察缝合线跟随移动
   - 注意耦合效果：缝合线会随器械移动但保持柔性

5. **调整参数**
   - 修改"重力"值观察下垂效果（-0.5到-2.0）
   - 修改"阻尼"值观察能量耗散（0.9到1.0）

6. **释放缝合线**
   - 点击"释放缝合线"按钮解除耦合

7. **重置仿真**
   - 点击"重置"按钮恢复初始状态

## 物理参数说明

| 参数 | 默认值 | 范围 | 说明 |
|-----|-------|-----|------|
| 时间步长 | 0.001 | 0.0001-0.01 | 仿真时间步长（秒） |
| 重力 | -0.5 | -2.0-0.0 | Y轴重力加速度（m/s²） |
| 阻尼 | 0.99 | 0.9-1.0 | 速度衰减系数 |

## 注意事项

1. **夹持距离** - 器械需要在0.05米范围内才能成功夹持
2. **耦合效果** - 缝合线会跟随器械移动，但会保持物理柔性
3. **性能** - 仿真线程运行在后台，不会影响UI响应
4. **关闭** - 直接关闭窗口会自动停止仿真线程

## 技术实现

- **物理引擎** - XPBD (Extended Position Based Dynamics)
- **约束求解** - 距离约束、耦合约束
- **渲染** - Matplotlib 3D绘图
- **UI框架** - PyQt5
- **架构** - Model-View-Controller + 多线程
