## 改进内容总结

### 1. 数值稳定性修复

**问题**: 原始版本出现 NaN（非数值）错误，导致仿真崩溃

**原因**:
- 粒子质量过小（0.01kg）而逆质量过大（100）
- 重力加速度过大（-9.81 m/s²）导致速度积累过快
- 约束求解器迭代次数过多，导致数值不稳定

**修复措施**:
- 调整物理参数：降低重力至 -0.5 m/s²（厘米级模拟）
- 优化约束求解：使用标准PBD过松弛因子 k=0.5
- 减少迭代次数：从15次降至10次
- 增加阻尼系数：从0.995提升至0.99

### 2. 多体耦合机制改进

**问题**: 器械与缝合线耦合失效，缝合线不随器械移动

**解决方案**:
- 引入**耦合约束（Coupling Constraint）**机制
- 夹持时创建硬约束，强制粒子跟随器械夹持点
- 每帧更新约束目标位置，实现实时耦合
- 释放时移除约束，传递速度保持物理连续性

**耦合流程**:
```
1. 器械夹持粒子 → 创建耦合约束
2. 器械移动 → 更新约束目标位置
3. 约束求解 → 强制粒子位置=夹持点位置
4. 器械释放 → 移除约束，传递速度
```

### 3. 约束求解器优化

**改进前**:
- 使用XPBD柔度参数，复杂且不稳定
- 弯曲约束导致额外计算开销

**改进后**:
- 简化距离约束求解
- 移除不稳定的弯曲约束
- 耦合约束最后求解（最高优先级）
- 使用子步积分提升稳定性
