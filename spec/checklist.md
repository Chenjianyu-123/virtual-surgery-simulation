# 虚拟手术缝合多体耦合仿真系统 - 检查清单

## 代码质量检查清单

### 编码规范
- [ ] 使用 C++17/20 标准
- [ ] 遵循统一的命名规范
  - [ ] 类名使用 PascalCase
  - [ ] 函数名使用 camelCase
  - [ ] 成员变量使用 m_ 前缀或下划线后缀
  - [ ] 宏定义使用全大写 + 下划线
- [ ] 所有头文件使用 `#pragma once` 或 include guard
- [ ] 避免使用裸指针，优先使用智能指针
- [ ] 避免内存泄漏，使用 RAII 模式

### 代码注释
- [ ] 所有公共接口都有文档注释
- [ ] 复杂算法有详细注释说明

### 错误处理
- [ ] 所有可能失败的操作都有错误检查
- [ ] 使用异常处理异常情况
- [ ] 返回错误码时检查返回值
